# clawithme 🕵️‍♂️

**Claw the open web，with me.**

输入一个用户名，发现 ta 在 3000+ 平台的踪迹 — 从社交网络到技术社区，从中国本地平台到全球站点。构建完整的公开身份全景图。

## 路线图

- [x] 继承 maigret 核心引擎（3000+ 站点检测）
- [ ] 扩充中国社交网络（Bilibili、小红书、抖音等）
- [ ] 加强 LinkedIn / 真实身份网络
- [ ] 深度公开信息爬取
- [ ] 全景报告（Geist 风格可视化）

## 快速开始

```bash
pip install -r requirements.txt
python -m clawithme <username>
```
